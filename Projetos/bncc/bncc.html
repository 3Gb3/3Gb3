Sobre o Projeto

Sobre o Projeto

O Projeto BNCC é uma aplicação desenvolvida em Python utilizando a biblioteca Tkinter para interface gráfica. A ferramenta foi criada para auxiliar educadores na identificação de códigos da Base Nacional Comum Curricular (BNCC). O sistema realiza a leitura completa de arquivos PDF, procura por códigos da BNCC e retorna os códigos encontrados junto com seus respectivos significados. Este projeto foi desenvolvido especialmente para auxiliar no planejamento educacional e organização de conteúdos pedagógicos.

Funcionalidades

Funcionalidades

  • Leitura e processamento completo de arquivos PDF
  • Busca automática por códigos da BNCC no documento
  • Exibição dos códigos encontrados com seus significados
  • Interface gráfica intuitiva desenvolvida com Tkinter
  • Sistema de análise rápida e eficiente
  • Facilita o trabalho de planejamento pedagógico

Objetivos do Projeto

Objetivos do Projeto

O objetivo principal foi criar uma ferramenta prática que facilitasse o trabalho de identificação e organização dos códigos da BNCC em documentos educacionais. O projeto surgiu de uma necessidade real no contexto educacional, tornando mais ágil o processo de planejamento e alinhamento de conteúdos com as diretrizes da Base Nacional Comum Curricular.
