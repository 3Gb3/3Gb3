Sabio - Bot de Discord

Python Discord.py Automation

Sobre o Projeto

Sabio é um bot de Discord desenvolvido em Python para o meu servidor pessoal. O bot foi criado para adicionar funcionalidades divertidas e úteis ao servidor, incluindo reprodução de músicas, exibição de memes, comando de SHIP para entretenimento dos membros, entre outras tarefas personalizadas. Desenvolvido utilizando Python e as bibliotecas de integração Python + Discord.

Funcionalidades

  • Sistema de reprodução de música no servidor
  • Comando para exibir memes aleatórios
  • Comando SHIP para combinações divertidas entre membros
  • Comandos personalizados para entretenimento
  • Automação de tarefas no servidor Discord
  • Interface de comandos simples e intuitiva

Tecnologias Utilizadas

  • Linguagem: Python
  • Bibliotecas: Discord.py e bibliotecas de integração
  • Foco: Entretenimento e automação para comunidades Discord

Objetivos do Projeto

O objetivo do Sabio é proporcionar entretenimento e funcionalidades úteis para o meu servidor Discord pessoal. O bot foi desenvolvido com foco em criar uma experiência mais dinâmica e divertida para os membros da comunidade, oferecendo recursos de música, memes e interações sociais através de comandos simples e personalizados.
