Sobre o Projeto

Sobre o Projeto

Sage é um bot para Discord desenvolvido em Python, criado para centralizar utilidades e automações no meu servidor pessoal. A base atual já contempla comandos personalizados e uma estrutura modular para facilitar futuras expansões. Neste momento, o desenvolvimento de novas funcionalidades está pausado, pois estou focado em um projeto de criação voltado para IA, que será integrado futuramente ao próprio Sage.

Desenvolvimento atualmente pausado

Funcionalidades

Funcionalidades

  • Comandos de entretenimento (memes e interações sociais)
  • Comando SHIP para combinações divertidas entre membros
  • Automação de tarefas recorrentes no servidor Discord
  • Estrutura de comandos personalizada em Discord.py
  • Base modular para expansão de novas funcionalidades
  • Organização de respostas e eventos para facilitar manutenção

Objetivos do Projeto

Objetivos do Projeto

O objetivo do Sage é evoluir para um bot mais inteligente, capaz de combinar automação, utilidade e interações contextualizadas para a comunidade. No curto prazo, meu foco está no desenvolvimento de um projeto de criação de IA dedicado. Após essa etapa, essa IA será implementada no próprio bot para ampliar suas capacidades.
